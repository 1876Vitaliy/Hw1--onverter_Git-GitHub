ДЗ 1. Git. GitHub

Створіть проект Converter, де реалізуйте функціонал конвертації миль в кілометри, кілометрів в милі.

Конвертації та виведення результатів в консоль мають бути реалізовані окремими методами.

Розробку вести під Git та GitHub.

Конвертацію миль в кілометри здійсніть у гілці master, в гілці develop додайте конвертацію кілометрів в милі.

Забезпечте злиття develop в master.

Всі зміни заливайте до свого GitHub репозиторію.

Посилання на свій GitHub репозиторій розмістіть в LMS.
